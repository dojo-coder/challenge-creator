# Contributing to DojoCode Challenge Creator

Thanks for your interest in contributing! This project helps creators build coding challenges for the [DojoCode](https://dojocode.io) platform.

## What We Accept

- Bug fixes and improvements to the tooling scripts
- Improvements to documentation
- Sample challenge contributions

## Reporting Bugs

Open an issue on [GitHub Issues](https://github.com/dojo-coder/challenge-creator/issues) with:

- A clear description of the problem
- Steps to reproduce
- Expected vs. actual behavior
- Your environment (OS, Node.js version, AI tool used)

## Pull Request Process

1. Fork the repo and create a feature branch from `main`
2. Make your changes
3. Ensure `npm install` runs without errors
4. Test any new or modified templates with the sample challenge workflow
5. Submit a PR with a clear description of the changes

## Challenge Structure Reference

`AGENTS.md` is the source of truth for challenge structure, file formats, and AI workflows. Please review it before making changes to templates or challenge-related tooling.

## Code of Conduct

Be respectful and constructive. We're building tools to help people learn to code — let's keep the community welcoming for everyone.
