Create an Image Slider using Vanilla TypeScript.

The function will receive an array of image URLs.

There should be one img tag that changes its source.

There should also be two buttons under the image: Previous and Next. On click of the buttons the img source will change, loading the next or previous image from the list of images.

The previous and next buttons should be disabled if no other images are available given the currently selected index, current image index is 0 or last image in array.

ImageSlider HTML Output

```
<div>
    <img width="200" height="250" alt="Image 1" />
    <button>Previous</button>
    <button>Next</button>
</div>
```
